git init : 저장소 초기화

git status : 저장소 상태

git add file : 버전 관리 file 등록

git config --global user.email "user_email" : git에 등록된 이메일

git config --global user.name "user_name" : git에서 사용하는 이름

git log -p : 변경사항 log

git log --stat : log의 통계를 보는것

git add --all : 변경사항 모두 추가

git commit -a -m "first files" : -a 모든파일, -m 코맨트(설명)
